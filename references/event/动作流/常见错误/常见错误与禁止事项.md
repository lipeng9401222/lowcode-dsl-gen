# 常见错误与禁止事项

**本文档汇总动作流 DSL 生成中的高频错误和禁止操作。**

> 本文档从 `event/动作流/index.md` 拆分而来，便于按需加载。

---

## ⚠️ 常见错误速查表

| 错误类型 | 错误示例 | 正确做法 |
|---------|---------|---------|
| **顶层结构顺序错误** | `workflow` 在 `app` 之前 | **必须按顺序：`app` → `workflow` → `kind` → `version` → `dependencies`** |
| **层次错误** | `app` 和 `workflow` 不平级 | `workflow` 必须在顶层，与 `app` 平级 |
| **层次错误** | `category` 和 `panelType` 位于 `data` 根级别 | `category` 和 `panelType` 必须位于 `actionData` 内部 |
| **字段遗漏** | 节点缺少 `position` 字段 | 参考基础结构文档，补全所有必需字段 |
| **连线错误** | 条件判断节点只有 `true` 分支 | 必须同时有 `true` 和 `false` 分支连线 |
| **资产滥用** | 用通用资产创建部门 | 必须使用 `query_action(query="部门")` 查询并使用 `methodName: "新增部门"` 或 `methodName: "新增部门信息"` |
| **Webhook 错误** | `ParentTaskUID: -1` | 必须使用字符串 UUID：`ParentTaskUID: "root"` |
| **编造字段** | 使用不存在的 `data.options` | 严格遵循文档中的字段定义 |
| **features配置错误** | `features: {}`（空配置） | 必须按照Workflow配置文档示例配置完整的features结构 |
| **⚠️ 迭代内节点缺少 iteration_id（最高频错误）** | 迭代内业务节点只有 `isInIteration: true` 但缺少 `data.iteration_id` 属性 | **必须在data层级添加 `iteration_id: "迭代节点ID"`（与isInIteration同级），这是导入必需字段，缺失会导致导入报错。示例：`data: { isInIteration: true, iteration_id: "1737450004000", ... }`** |
| **迭代开始节点配置不完整** | `custom-iteration-start` 类型节点缺少 `parentId`、`selectable: false`、`draggable: false`、`data.isInIteration: true` 等字段 | 必须补充所有必需字段：`data.isInIteration: true`、`selectable: false`、`draggable: false`、`parentId`（指向迭代节点）、`sourcePosition: "right"`、`targetPosition: "left"`、`width: 44`、`height: 48`、`zIndex: 1002` |
| **迭代开始连线属性缺失** | 迭代开始节点连线缺少 `isInIteration`、`iteration_id` 属性 | 必须添加 `data.isInIteration: true`、`data.iteration_id: "迭代节点ID"`、`data.isInLoop: false` |
| **主迭代节点连线sourceHandle错误** | 主迭代节点连线（从迭代节点到外部）的 `sourceHandle` 设置为 `"true"` | 主迭代节点连线必须设置 `sourceHandle: "source"`（⚠️ 不是 `"true"`） |
| **迭代开始连线isInLoop错误** | 迭代开始节点连线的 `isInLoop` 设置为 `true` | 迭代开始节点连线必须设置 `isInLoop: false`（注意与普通迭代内连线不同） |
| **条件判断节点cases缺少id** | `cases[0]` 只有 `case_id: "true"`，缺少 `id: "true"` | 必须添加 `id: "true"` 字段 |
| **条件判断节点conditions缺少必需字段** | `conditions[i]` 缺少 `id` 或 `value` 字段 | 必须添加 `id`（UUID格式）和 `value`（可为空字符串 `""`） |
| **条件判断节点字段顺序错误** | `data` 下先定义 `type`，后定义 `cases` | **必须先定义 `cases`，后定义 `type`**（字段顺序会影响解析） |
| **资产节点参数顺序错误** | 资产节点的参数顺序与场景示例或面板配置文档不一致 | **优先按照场景示例或面板配置文档中的参数顺序配置**，无文档时才参考 `query_action` 结果 |
| **⚠️ 数据表资产参数名错误（高频）** | `tableName: "xxx"` | **必须使用 `sqlTableName: "xxx"`**（参考场景示例，而非 query_action 返回） |
| **⚠️ 数据表资产冗余参数 appGuid** | 添加了 `appGuid: ""` | **数据表资产不需要此参数，必须删除**（场景示例中无此参数） |
| **⚠️ 查询详情冗余参数 sortFields** | 查询详情节点添加 `sortFields: []` | **查询详情（misfindaction）不需要此参数，必须删除** |
