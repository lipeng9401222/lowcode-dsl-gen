# 视图模型速查

> 本文档从 `页面设计器/index.md` 拆出，包含 models（数据模型）定义与 children（视图树）节点结构速查。

## models（数据模型）

只声明业务数据模型，UI 临时状态（loading/选中行）不放这里。

```json
{
  "models": {
    "search": {
      "type": "record",
      "fields": {
        "keyword": {
          "type": "string",
          "label": "关键词"
        }
      }
    },
    "employee": {
      "type": "record",
      "primaryKey": "id",
      "fields": {
        "id": { "type": "string", "label": "ID" },
        "name": {
          "type": "string",
          "label": "姓名",
          "rules": { "required": true }
        }
      },
      "initial": {
        "name": "",
        "deptId": null
      }
    },
    "employeeList": {
      "type": "collection",
      "primaryKey": "id",
      "fields": {
        "id": { "type": "string" },
        "name": { "type": "string", "label": "姓名" }
      }
    }
  }
}
```

### 模型类型

| type | 含义 |
|------|------|
| `record` | 单条业务数据（表单/查询条件/详情） |
| `collection` | 多条业务数据（列表/子表/重复组） |

### 字段元信息

| 字段 | 说明 |
|------|------|
| `type` | 基础类型：string/number/boolean/date/object/array |
| `label` | 字段显示名 |
| `rules` | 字段级规则（required/readonly/submit/mask/validate） |

### 字段 rules

```json
{
  "rules": {
    "required": true,
    "readonly": false,
    "submit": true,
    "mask": false,
    "validate": [
      {
        "type": "email",
        "message": "请输入正确的邮箱",
        "when": "notEmpty(model.employee.email)"
      }
    ]
  }
}
```

支持的 validate 规则：
- `type`：预设校验规则类型（如 `email`、`number`）
- `expression`：表达式校验
- `validator`：引用顶层 actions 中的自定义校验
- `options`：规则配置对象
- `when`：表达式，true 时该校验生效
- `message`：错误提示

## children（视图树）

视图树由 ViewNode 数组组成。

### ViewNode 字段

```json
{
  "id": "employeeName",
  "version": "1.0.0",
  "type": "input",
  "model": "employee.name",
  "textModel": "employee.deptName",
  "source": "deptOptions",
  "props": { "label": "姓名" },
  "style": {},
  "events": {},
  "visible": "model.employee.enabled == true",
  "children": []
}
```

| 字段 | 说明 |
|------|------|
| `id` | 节点稳定标识 |
| `version` | 节点版本号 |
| `type` | 运行态组件类型（不枚举具体值） |
| `model` | 主数据绑定（短路径，如 `employee.name`） |
| `textModel` | 文本侧绑定（用于下拉、树选择的 text/value 对） |
| `source` | 资源 id（引用顶层 resources） |
| `props` | 组件语义配置 |
| `style` | 显式指定的样式 |
| `events` | 事件 → action 引用 |
| `visible` | 视图级显隐（布尔或表达式） |
| `children` | 子节点 |

### 常见组件 type（仅参考，运行时定义）

| type | 用途 |
|------|------|
| `input` / `text` / `textarea` | 文本输入/展示 |
| `select` / `treeSelect` | 选择器 |
| `button` | 按钮 |
| `table` | 表格（带 `columns` 根属性） |
| `grid` | 网格布局 |
| `divider` | 分隔线 |
| `upload` | 文件上传 |
| `form` | 表单容器 |

### table 节点（特殊）

`columns` 是 table 的根属性，不放在 `props` 或 `children` 里：

```json
{
  "type": "table",
  "model": "employeeList",
  "columns": [
    { "type": "field", "field": "name", "label": "姓名" },
    {
      "type": "actions",
      "label": "操作",
      "items": [{ "label": "编辑" }]
    }
  ]
}
```
