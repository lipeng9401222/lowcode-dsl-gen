# 03 资产拆分

目标：把应用蓝图拆成应用根下的 8 类资产。

## 映射规则

| 蓝图内容 | 资产类型 |
|----------|----------|
| 应用名称、apptag、套件 | `appinfo.lowcode.yml` |
| 引用其他应用的字典/表/页面 | `appref.lowcode.yml` |
| 枚举、状态、类型 | `codeitem/*.yml` |
| 业务对象、字段 | `mis/*.yml` |
| 菜单、路由入口 | `module/*.module.yml` |
| 接口编排、动态 API | `event/*.event.yml` |
| 审批节点、材料、按钮 | `workflow/*.workflow.yml` |
| 列表、表单、详情页 | `pagedesigne/*.pagedesigne.yml` |

## 拆分输出

每个资产至少记录：

- 资产类型
- 资产名称
- 目标文件名
- 生成方式（脚本 / 模板 / 手工 JSON）
- 依赖关系（如 mis 字段引用 codeitem、页面调用 endpoint）
