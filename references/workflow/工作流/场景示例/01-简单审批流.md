# 简单审批流示例

> 最小骨架：开始 → 申请 → 审批 → 结束 + 浏览（孤立）

## 使用脚本一键生成

```bash
python scripts/add_workflow.py \
  --app-root /path/to \
  --name "请假申请流程" \
  --material "请假申请表"
```
