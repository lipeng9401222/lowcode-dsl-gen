# 流程版本详情（workflowProcessVersion）

> 本文档定义 workflowProcessVersion 的完整字段，含 工作流扩展字段。

## workflowProcessVersion（流程版本详情）

```yaml
workflowProcessVersion:
  processversionguid: <版本UUID>     # 与所有子节点一致
  processguid: <流程UUID>            # 与 workflowProcess.processguid 一致
  processversionname: "请假申请流程20260429"
  version: V1
  status: 10
  designversion: senior
  direction: '90'                    # '0':水平；'90':垂直（字符串）
  createdate: 2026-01-12T08:42:04Z
  updatedate: 2026-04-29T00:30:17Z
  author: <创建人 UUID>
  # 工作流扩展字段
  isshowlinegraph: Normal             # 'Normal'(直线)、'Orthogonal'(折线)
  isshownodesimple: details           # 'details'(详情)、'simple'(精简)
  revokeoption: 20                    # 10(不允许)、20(撤销尚未办理)、30(撤销尚未通过)、40(按天数限制)
  revokeallowday: 0                   # revokeoption=40 时填天数
  revokeremindoption: 20              # 10(知会撤销)、20(静默撤销)
  noparticipatoroption: 10            # 10(不处理)、20(转部门管理员)、30(指定人员审批)
  defaultuserguid: null               # noparticipatoroption=30 时填处理人 UUID
```

## workflowProcessVersion 字段表

| 字段 | 含义 | 类型 / 取值 | 必填 |
|------|------|-------------|------|
| `processversionguid` | 版本唯一标识 | UUIDv4 | 是 |
| `processguid` | 关联流程标识 | 同 `workflowProcess.processguid` | 是 |
| `processversionname` | 版本名称 | 通常同流程名称 | 是 |
| `version` | 版本号 | 如 `V1` / `V2`，默认 `V1` | 是 |
| `status` | 版本状态 | `10` 启用 / `20` 停用，默认 `10` | 是 |
| `designversion` | 版本类型 | `senior` / `simple`，默认 `senior` | 是 |
| `direction` | 排版方向 | 字符串 `'0'` 水平 / `'90'` 垂直 | 是 |
| `createdate` | 创建时间 | `YYYY-MM-DD HH:MM:SS` 或平台兼容时间字符串 | 是 |
| `updatedate` | 更新时间 | 同上 | 是 |
| `author` | 创建人标识 | UUIDv4 | 是 |
| `isshowlinegraph` | 连线样式 | `Normal` / `Orthogonal` | 是 |
| `isshownodesimple` | 节点显示方式 | `details` / `simple` | 是 |
| `revokeoption` | 发起人撤销设置 | `10` 不允许 / `20` 撤销尚未办理 / `30` 撤销尚未通过 / `40` 按天数 | 是 |
| `revokeallowday` | 允许撤销天数 | `revokeoption=40` 时有效，默认 `0` | 否 |
| `revokeremindoption` | 撤销提醒设置 | `10` 知会撤销 / `20` 静默撤销 | 否 |
| `noparticipatoroption` | 预设处理人不存在时操作 | `10` 不处理 / `20` 转部门管理员 / `30` 指定人员 | 是 |
| `defaultuserguid` | 指定处理人标识 | `noparticipatoroption=30` 时填 UUID，否则 `null` | 否 |

## 工作流扩展字段速查

| 字段 | 含义 | 类型 | 合法值 |
|------|------|------|--------|
| `isshowlinegraph` | 连线样式 | 字符串 | `'Normal'`（直线）、`'Orthogonal'`（折线） |
| `isshownodesimple` | 节点显示 | 字符串 | `'details'`（详情）、`'simple'`（精简） |
| `revokeoption` | 撤销选项 | 整数 | `10`（不允许）、`20`（撤销尚未办理）、`30`（撤销尚未通过）、`40`（按天数） |
| `revokeallowday` | 撤销天数 | 整数 | revokeoption=40 时有效 |
| `revokeremindoption` | 撤销提醒 | 整数 | `10`（知会撤销）、`20`（静默撤销） |
| `noparticipatoroption` | 无处理人时 | 整数 | `10`（不处理）、`20`（转部门管理员）、`30`（指定人员） |
| `defaultuserguid` | 指定处理人 | UUID/null | noparticipatoroption=30 时填 |
