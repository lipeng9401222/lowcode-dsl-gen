# 外部方法与事件配置

> 本文档定义流程外部方法（method / workflowMethodParameter）及事件配置（workflowEvent）。
> 外部方法提供业务逻辑入口，事件配置通过引用外部方法实现流程流转时的自动化操作。

---

## § 1 外部方法（method / workflowMethodParameter）

### method（流程外部方法）

定义流程中需要执行的业务方法。1 个版本可关联 ≥0 个外部方法。

| 字段 | 含义 | 类型 | 必填 | 基础/工作流扩展 |
|------|------|------|------|-------------------|
| `methodGuid` | 方法唯一标识 | UUID | 是 | 工作流扩展 |
| `processversionguid` | 版本标识 | UUID | 是 | 工作流扩展 |
| `dllpath` | 路径 | 字符串 | 否 | 工作流扩展 |
| `typename` | 外部方法所在类路径 | 字符串（如 `com.epoint.ztb.workflow.ExternalMethods`） | 是 | 工作流扩展 |
| `methodname` | 方法名称 | 字符串（如 `updateRecordStatus`） | 是 | 工作流扩展 |
| `returnvaluetype` | 返回值类型 | `Void`/`boolean`/`String` 等 | 是 | 工作流扩展 |
| `note` | 备注 | 字符串（描述方法用途） | 是 | 工作流扩展 |
| `ordernum` | 排序 | ≥0 | 是 | 工作流扩展 |
| `ruleguid` | 规则标识 | 动作流标识（可填动作流 `app.id` / `rowguid` / `sign`） | 否 | 工作流扩展 |

### workflowMethodParameter（方法参数）

每个外部方法含 ≥0 个参数。

| 字段 | 含义 | 类型 | 必填 | 基础/工作流扩展 |
|------|------|------|------|-------------------|
| `mpGuid` | 参数唯一标识 | UUID | 是 | 工作流扩展 |
| `methodGuid` | 所属方法标识 | UUID | 是 | 工作流扩展 |
| `mpname` | 参数名称 | 字符串（如 `context`、`status`） | 是 | 工作流扩展 |
| `mptype` | 参数类型 | `10`（字符型）、`20`（整形）、`30`（数字）、`40`（时间）、`50`（布尔）、`100`（对象） | 是 | 工作流扩展 |
| `mpvalue` | 参数值 | 字符串（如 `[#=ExecutionContext#]`、`2`） | 是 | 工作流扩展 |
| `encrypt` | 是否加密 | `'0'`（不加密）、`'1'`（加密） | 否 | 工作流扩展 |
| `processversionguid` | 版本标识 | UUID | 是 | 工作流扩展 |
| `orderNum` | 排序号 | ≥0 | 是 | 工作流扩展 |
| `mpnamedescription` | 参数名描述 | 字符串 | 否 | 工作流扩展 |
| `mpvaluedescription` | 参数值描述 | 字符串 | 否 | 工作流扩展 |

### 示例

```yaml
method:
  - methodGuid: 007361cb-dab8-49b2-9a3e-93238086288c
    processversionguid: <版本UUID>
    dllpath: jsgcztb.outdll\JSGCZtb.OutDll.dll
    typename: com.epoint.ztb.workflow.ExternalMethods
    methodname: updateRecordStatus
    returnvaluetype: Void
    note: "更新为待审核状态"
    ordernum: 84
    workflowMethodParameter:
      # 常量参数
      - mpGuid: <UUID>
        methodGuid: 007361cb-dab8-49b2-9a3e-93238086288c
        mpname: status
        mptype: 10
        mpvalue: 2
        encrypt: 0
        processversionguid: <版本UUID>
        ordernum: 50
      # 工作流上下文参数
      - mpGuid: <UUID>
        methodGuid: 007361cb-dab8-49b2-9a3e-93238086288c
        mpname: context
        mptype: 10
        mpvalue: "[#=ExecutionContext#]"
        encrypt: 0
        processversionguid: <版本UUID>
        ordernum: 50
        mpnamedescription: "上下文对象参数(仅支持单独配置)"
```

## § 2 事件配置（workflowEvent）

定义流程流转过程中预置业务操作的事件。1 个版本可关联 ≥0 个事件。

事件通过 `belongTo` + `eventType` 组合确定触发时机：

| `belongTo` | 含义 | 可用 `eventType` |
|-----------|------|--------------------|
| `20` | 活动实例事件 | `10`（创建前）、`15`（创建后）、`60`（签收后）、`20`（完成前）、`25`（完成后） |
| `22` | 活动操作事件 | `50`（操作前）、`55`（操作后） |

> **关联说明**：事件可通过 `eventMethodGuid` 引用 [§ 1 外部方法](#§-1-外部方法method--workflowmethodparameter) 中定义的 `method.methodGuid`，也可通过 `ruleGuid` 直接引用动作流。二者至少配置一个；同时配置时以项目运行时约定为准。

| 字段 | 含义 | 类型 | 必填 | 基础/工作流扩展 |
|------|------|------|------|-------------------|
| `eventGuid` | 事件唯一标识 | UUID | 是 | 工作流扩展 |
| `eventName` | 事件名称 | 字符串 | 是 | 工作流扩展 |
| `belongTo` | 事件来源 | `20`（活动实例）、`22`（活动操作） | 是 | 工作流扩展 |
| `eventType` | 事件类型 | 见上表 | 是 | 工作流扩展 |
| `eventMethodGuid` | 外部方法标识 | UUID（指向 [`method.methodGuid`](#method流程外部方法)） | 否（与 `ruleGuid` 至少一个） | 工作流扩展 |
| `ruleGuid` | 规则标识 | 动作流标识（可填动作流 `app.id` / `rowguid` / `sign`） | 否（与 `eventMethodGuid` 至少一个） | 工作流扩展 |
| `sourceGuid` | 数据来源标识 | 活动实例→活动标识；操作→按钮标识 | 是 | 工作流扩展 |
| `processversionguid` | 版本标识 | UUID | 是 | 工作流扩展 |
| `syncType` | 同步类型 | `0`（同步）、`1`（异步） | 是 | 工作流扩展 |
| `orderNumber` | 排序号 | 数字，默认 `100` | 是 | 工作流扩展 |

### 示例

```yaml
workflowEvent:
  # 按钮操作后事件：送下一步按钮操作后更新状态
  - eventGuid: <UUID>
    eventName: "更新业务表状态字段为0（送下一步按钮操作后处理）"
    belongTo: 22          # 活动操作
    eventType: 55         # 操作后
    sourceGuid: <按钮 operationguid>
    processversionguid: <版本UUID>
    syncType: 0
    orderNumber: 100
    eventMethodGuid: <外部方法 methodGuid>
  # 活动实例事件：当前活动实例完成后处理
  - eventGuid: <UUID>
    eventName: "更新业务表A字段为B（活动实例完成后处理）"
    belongTo: 20          # 活动实例
    eventType: 25         # 完成后
    sourceGuid: <活动 activityguid>
    processversionguid: <版本UUID>
    syncType: 0
    orderNumber: 100
    eventMethodGuid: <外部方法 methodGuid>
```
