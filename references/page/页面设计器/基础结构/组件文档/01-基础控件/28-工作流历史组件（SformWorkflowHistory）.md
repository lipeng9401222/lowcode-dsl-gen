# SformWorkflowHistory 工作流历史组件

## 组件选择摘要

- 适用场景：展示流程审批历史、流转记录。
- 优先使用：详情页或审批页需要查看流程轨迹。
- 不建议用于：普通业务日志或表格数据。
- 读取建议：生成页面时先根据本摘要判断是否命中；命中后再继续读取下方属性、事件、方法和示例。

![](https://oa.epoint.com.cn/h5/fileattaches/20260624/1782268441213_664d4783/sform-workflow-history.png)

## 属性 (Props)

该组件无自定义属性，数据通过 pageConfig.workflow 自动获取。

## 事件 (Events)

无
